James Ortega

Role:

  • James is a user wanting to log into Toss Me a Game

Technologies:

  • IPhone 13 Pro max
  • Asus Laptop running Windows 11
  • James is an average user and proficient with technology. He knows how to get around the web with no problem.

Behaviors:

  • James is always looking for a way to connect to people that like games.
  • James likes to navigate the internet safely.
  • James cares about the internet safety of others.

Attitudes:

  • James is a caring individual, whether it be family, friends, or strangers.
  • James has a passion for video games, or any type of games that brings him entertainment.
  • He is a visual person in which he does not like reading long text.

Scenario

  • James is looking for a safe way where he can view games that are listed online.

Frustration

  • He does not like to be on websites where it is anonymous.
  • James does not want to navigate a site where all products are listed. He prefers to only find game listings.
  • He does not like a distracting layout on websites that he visits.

Goals:

  • Find a game listing website where there is a pleasant community.
  • James wants to be able to have a profile where he can sign in and out.
  • Navigate an aesthetically clean website with a focus on visuals.

User Story:

  • As a user of this app, I want to be able to create an account.

Use Case and Interaction Flow

Use Case:

  • James gets wind of a web app that has free listing of games.

Pre-Condition:

  • James opens up a web browser.

Post-Condition:

  • James creates a profile that allows him to log in.

Interaction Flow

  • James opens up his browser and clicks google search bar
  • Google prompts him to search.
  • James searches Toss Me a game and clicks on the website
  • Toss Me a Game is displayed in his browser
  • James clicks on the profile icon
  • Toss Me a Game redirects him to the sign in page
  • James clicks on the "Create Account" button
  • Toss Me a Game redirects him to the "Create Profile Page"
  • James enters his first name, last name, email and password. He chooses not to upload a profile picture
  • Toss Me a Game creates his personal account.
